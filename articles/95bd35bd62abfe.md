# そもそも変数とは
データを保存しておくための箱。
わかりやすく言うと、料理。
塩や醤油、肉や魚がそれぞれの容器に入っていて、必要になったら使うイメージ。

# 変数宣言の種類
- var
- let
- const

const：再代入× 再定義×
let：再代入○ 再定義×
var：再代入○ 再定義○
<br>
基本的には`const`を使用し、どうしようもなければ`let`を使用する。
`var`は使ったら負けと思っておく。
なぜなら、なんでもOKになってしまうため、エラーやバグが発生しやすくなるため。
